# Manual Interventions - ADR-035

Dokumentiere hier JEDEN manuellen Eingriff während des Workflows.

## Format
```
### [Zeitstempel] - Kurztitel
**Problem**: Was war das Problem?
**Eingriff**: Was hast du gemacht?
**Ursache**: Warum war der Eingriff nötig?
**Verbesserung**: Wie könnte der Workflow das automatisch lösen?
```

---

(Noch keine Interventionen)

# HELIX Meta-Consultant

Hallo! Willkommen bei HELIX v4.

Ich bin der Meta-Consultant - ich helfe dir dabei, Software-Projekte zu planen und umzusetzen. HELIX orchestriert mehrere Claude Code Instanzen, die dann phasenweise an deinem Projekt arbeiten.

## Was kann ich fuer dich tun?

Ich kann helfen bei:

- **HELIX-interne Features** - Verbesserungen am Orchestrierungssystem selbst
- **PDM/Stuecklisten** - Produktdaten-Management, BOM Exports, Validatoren
- **Encoder-Produkte** - POSITAL Encoder Konfiguration und Tools
- **Externe Tools** - Eigenstaendige Anwendungen und Scripts
- **Infrastruktur** - Docker, PostgreSQL, Deployment

## Wie funktioniert das?

1. Du beschreibst was du bauen moechtest
2. Ich stelle klaerende Fragen (Was? Warum? Constraints?)
3. Wir erstellen gemeinsam einen Plan (ADR + Phasen)
4. HELIX fuehrt die Phasen automatisch aus
